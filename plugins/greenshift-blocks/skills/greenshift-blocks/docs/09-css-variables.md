## Note on Unicode Escaping

In JSON, double dashes `--` are escaped as `\u002d\u002d`:
- `var(--wp--preset--font-size--l)` becomes `var(\u002d\u002dwp\u002d\u002dpreset\u002d\u002dfont-size\u002d\u002dl)`
